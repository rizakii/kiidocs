Location/apps/{appID}/topics/{topicID}/push/subscriptions/users/{userID}/configuration

header

header
Required: Yes

header
Required: Yes

PUT

/apps/{appID}/topics/{topicID}/push/subscriptions/users/{userID}/configuration

Description

Update the push configuration of the subscription of the accessed user.

Security

Can be performed by the administrator and the subscribed user itself.

Content-Typeapplication/vnd.kii.PushSubscriptionConfigurationModificationRequest+json

The new configuration data to update in the subscription.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
Params
Name Required? type Description
gcmufe:GCMConfigurationType
apnsufe:APNConfigurationType
sendAppIDbooleanSet true or false to send "AppID" field in notifications.
sendSenderbooleanSet true or false to send "Sender" field in notifications.
sendWhenbooleanSet true or false to send "When" field in notifications.
sendObjectScopebooleanSet true or false to send "ObjectScope" field in notifications.
sendTopicIDbooleanSet true or false to send "TopicID" field in notifications.
sendOriginbooleanSet true or false to send "Origin" field in notifications.
sendBucketIDbooleanSet true or false to send "BucketID" field in notifications.
sendBucketTypebooleanSet true or false to send "BucketType" field in notifications.
sendObjectIDbooleanSet true or false to send "ObjectID" field in notifications.
sendModifiedAtbooleanSet true or false to send "ModifiedAt" field in notifications.
sendMessageTypebooleanSet true or false to send "MessageType" field in notifications.
