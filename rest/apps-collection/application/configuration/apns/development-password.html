PUT

/apps/{appID}/configuration/apns/development-password

Description

Set or change the development password.

Security

Only can be performed with administrator credentials.

Content-Typetext/plain

The password for development.

X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app

204

Content-Type

The password has been registered successfully.
