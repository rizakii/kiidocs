PUT

/apps/{appID}/configuration/gcm

Description

Set or change the GCM key.

Security

Only can be performed with administrator credentials.

Content-Typeapplication/vnd.kii.GCMKeyRegistrationRequest+json

All the required data to register the GCM key.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
Params
Name Required? type Description
keystring The GCM key
Sample Request
{ "key": "[string]" }
Response
201

Content-Type

The GCM key has been created successfully.

Content
NameType Description
Locationxsi:stringURI to the created GCM key
204

Content-Type

The GCM key has been changed successfully.

401
