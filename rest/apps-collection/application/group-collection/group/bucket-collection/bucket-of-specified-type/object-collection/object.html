This resource represents an object in the bucket.

The maximum size allowed for an object is 65534 bytes, and the fields of an object with a key name bigger than 250 bytes or an string value bigger than 190 bytes, will not be indexed, so it will not be possible to query for them.

When you are trying to create a new scope data for your app, you must consider some facts to create an appropiate bucket name. Following this pattern you can create a correct name for your bucketID: "a-zA-Z0-9_-". Also, this pattern has a length limitations between 2 and 64 characters.

DELETE

/apps/{appID}/groups/{groupID}/buckets/{bucketType}:{bucketID}/objects/{objectID}

Description

Delete the accessed object. Delete is conditional if "If-Match" header is provided, in that case the object will be deleted if the version is not stale. If "If-None-Match" header is provided the delete operation will be unconditional, the same if none of those headers are in the request. "If-Match" and "If-None-Match" headers can not be provided at the same time

Security

This action can be performed with administrator credentials, if not, the permission to perform this action depends on the ACL of the application, user or group, where the object of the bucket belongs.

PUT

/apps/{appID}/groups/{groupID}/buckets/{bucketType}:{bucketID}/objects/{objectID}

Description

Replace or create an object with the specified ID by template. If "If-Match" header is provided the operation is an update conditional, the object will be replaced if the version is not stale. If "If-None-Match" header is provided the replace operation will be unconditional. If none of those headers are in the request the operation will be an unconditional replace if the object exists or the object will be created if not exists. "If-Match" and "If-None-Match" headers can not be provided at the same time

Security

This action can be performed with administrator credentials, if not, the permission to perform this action depends on the ACL of the application, user or group, where the object of the bucket belongs.

PATCH

/apps/{appID}/groups/{groupID}/buckets/{bucketType}:{bucketID}/objects/{objectID}

Description

Update partially the accessed object, fields can be added or updated, but not deleted. If "If-Match" header is provided the operation is an update conditional, the object will be updated if the version is not stale. If "If-None-Match" header is provided the update operation will be unconditional, the same if none of those headers are in the request. "If-Match" and "If-None-Match" headers can not be provided at the same time
