This resource represents a group belonging to an application in Kii Platform.
Location/apps/{appID}/groups/{groupID}
  • GET
  • DELETE

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: Yes

GET

/apps/{appID}/groups/{groupID}

Description

Get the data of the current group.

Anonymous access is not allowed for this method.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
URL Params
Name Required? type Description
groupIDYesgroupID of the group being accessed
Response
200

Content-Type application/vnd.kii.GroupRetrievalResponse+json

The data of the group has been retrieved successfully.

Content
NameType Description
groupIDstringThe ID of the group
namestringThe name of the group
ownerstringThe ID of the user that owns the group
404

Content-Type application/vnd.kii.GroupNotFoundException+json

Content
NameType Description

Content
