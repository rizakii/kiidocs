GET

/apps/{appID}/groups

Description

Get a list of groups filtered by membership or ownership. One of the two query parameters must be provided, if "is_member" is provided it will be retrieved a list of groups in which the user is member, if "owner" is provided it will be retrieved the list of groups owned by the user.

Security

Anonymous access is not allowed for this method.

POST

/apps/{appID}/groups

Description

Create a new group. The owner will be added also as member by default.

Security

Anonymous access is not allowed for this method. If accessing with administrator credentials owner is not required for the group.

Content-Typeapplication/vnd.kii.GroupCreationRequest+json
