POST

/apps/{appID}/users

Description

Create a new user in the current application. In Kii Platform users the login is unique key, there can not be two users in the same application with the same login. Phone number and email are also unique keys only if they are verified. These two fields are verified if the values for "emailAddressVerificationRequired" and "phoneNumberVerificationRequired" in the application are set to false when the user is created, otherwise the user can verify them afterwards.

Content-Typeapplication/vnd.kii.RegistrationRequest+json or application/json
