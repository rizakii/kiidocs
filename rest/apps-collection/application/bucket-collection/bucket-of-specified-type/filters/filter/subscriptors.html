POST

/apps/{appID}/buckets/{bucketType}:{bucketID}/filters/{filterID}/push/subscription/users

Description

Subscribe the current authenticated user to the filter.

Security

Only can be performed with user credentials. The user needs query access to the bucket where the filter belongs.
