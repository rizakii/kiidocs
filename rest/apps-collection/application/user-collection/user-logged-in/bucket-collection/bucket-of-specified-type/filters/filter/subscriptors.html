POST

/apps/{appID}/users/me/buckets/{bucketType}:{bucketID}/filters/{filterID}/push/subscriptions/users

Description

Subscribe the current authenticated user to the filter. If bucket wouldn't exist it will be created before and the subscriber will become the bucket owner.

Security

Only can be performed with user credentials. The user needs query access to the bucket where the filter belongs and needs bucket creation permissions if bucket wouldn't exist.

POST

/apps/{appID}/users/me/buckets/{bucketType}:{bucketID}/filters/{filterID}/push/subscriptions/users

Description

Subscribe the current authenticated user to the filter with a push configuration. If bucket wouldn't exist it will be created before and the subscriber will become the bucket owner.

Security

Only can be performed with user credentials. The user needs query access to the bucket where the filter belongs and needs bucket creation permissions if bucket wouldn't exist.

Content-Typeapplication/vnd.kii.CreatePushSubscriptionRequest+json

The data to configure the parameters for the push messages the user will receive through this subscription.
