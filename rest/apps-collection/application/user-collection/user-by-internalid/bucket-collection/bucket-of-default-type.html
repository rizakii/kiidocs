--- layout: rest-api title: 'Bucket of default type' tags: rest-api ---
Access a bucket of the default backend type. This is the same than default:{bucketID}
Location/apps/{appID}/users/id:{internalID}/buckets/{bucketID}
  • GET
  • HEAD
  • DELETE

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: No

GET

/apps/{appID}/users/id:{internalID}/buckets/{bucketID}

Description

Retrieve a bucket

Security

This action can be performed with administrator credentials, if not, the permission to perform this action depends on the ACL of the application, user or group, where the bucket belongs.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
URL Params
Name Required? type Description
bucketIDYesbucketID of the bucket being accessed
Response
200

Content-Type application/vnd.kii.BucketRetrievalResponse+json

The bucket's data

The type of the bucket
