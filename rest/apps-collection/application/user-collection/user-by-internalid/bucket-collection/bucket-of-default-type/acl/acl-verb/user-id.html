GET

/apps/{appID}/users/id:{internalID}/buckets/{bucketID}/acl/{ACLVerb}/UserID:{subjectUserID}

Description

Check if this user has granted the verb for the accessed resource.

Security

The administrator always can perform this action and depending on the resource can also do it:

  • User: the user.
  • Group: the group owner.
  • Bucket: the scope-owner and the bucket creator.
  • Object: the scope-owner and the object owner.
  • Topic: the scope-owner and the topic creator.
