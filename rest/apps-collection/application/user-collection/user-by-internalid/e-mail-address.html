This resource represents the e-mail address of the user.
Location/apps/{appID}/users/id:{internalID}/email-address
  • GET
  • PUT
  • DELETE

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: No

GET

/apps/{appID}/users/id:{internalID}/email-address

Description

Get the e-mail address of the user.

Security

Anonymous access is not allowed. This can be performed by the administrator, the user itself and, if the property "exposeFullUserDataToOthers" of the application is set to "true", any user of the application can access also.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
Response
200

Content-Type application/vnd.kii.EmailAddressRetrievalResponse+json

The e-mail address has been retrieved successfully.

Content
NameType Description
emailAddressstringThe e-mail address
verifiedboolean Indicates whether the e-mail address is verified or not

PUT

/apps/{appID}/users/id:{internalID}/email-address

Description

Update the e-mail address of the user

Security

This method can be performed by the administrator and by the user itself, but the parameter "verified" can only be set to "true" with administrator credentials.

Content-Typeapplication/vnd.kii.EmailAddressModificationRequest+json

Params

verifiedbooleanIndicates whether the new e-mail address is already verified
