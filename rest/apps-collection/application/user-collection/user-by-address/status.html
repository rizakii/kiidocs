This represents the status of the user, it can be enabled or disabled. By default a user is enabled, changing the status to disabled implies the user can not login in Kii Platform, so it can not do any action, and will not receive any push notification either, this does not affect the data in the scope of the user or created by it, the data will remain accessible for the other users if they have the right permissions, as well as groups or topics created by the disabled user.
