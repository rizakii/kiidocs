PUT

/apps/{appID}/users/{accountType}:{address}/buckets/{bucketType}:{bucketID}/objects/{objectID}/body

Description

Add or replace the body of the object being accessed. This operation can be conditional if "If-Match" header is provided, in that case the object will added or replaced the body if the version is not stale. If "If-None-Match" header is provided this operation will be unconditional, the same if none of those headers are in the request "If-Match" and "If-None-Match" headers can not be provided at the same time.

Security

This action can be performed with administrator credentials, if not, the permission to perform this action depends on the ACL of the application, user or group, where the object of the bucket belongs.
