This resource represents a user as a subscriptor to the current topic.
Location/apps/{appID}/users/{userID}/topics/{topicID}/push/subscriptions/users/{userID}

PUT

/apps/{appID}/users/{userID}/topics/{topicID}/push/subscriptions/users/{userID}

Description

Subscribe this user to the current topic.

Security

Can be performed by the administrator and the user to subscribe itself.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
URL Params
Name Required? type Description
userIDYesuserID of the subscriptor user
Response
204

Content-Type

The user has been subscribed successfully.

404

Content-Type application/vnd.kii.UserNotFoundException+json

Happens when accessing a topic of a user and the user is not found, or when the user to subscribe is not found.
