This resource represents a user belonging to an application in the Kii platform. The user is accessed by one of its addresses: e-mail, phone number or login name.
Location/apps/{appID}/users/{accountType}:{address}
  • GET
  • POST
  • DELETE

Request Headers (Applies to all methods)

header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: No

GET

/apps/{appID}/users/{accountType}:{address}

Description

Retrieve the user information.

Security

Anonymous access to this method is not allowed. The access with credentials of administrator or the user itself will retrieve the full user's data. If the access is done with credentials of another user of the same application it depends on the property "exposeFullUserDataToOthers" of the application, if it is set to "true" the access will recover all the user's data, if it is set to "false" or "null" only will be recovered the fields "userID", "loginName" and "displayName".

POST

/apps/{appID}/users/{accountType}:{address}

Description

Modify user's data. In this method can it be modified the country, the displayName and the arbitrary fields of the user. The country and the displayName will eb modified if they have value in the request, if not they will remain untouched. For the arbitrary fields is different, they always are replaced with the content of the request, so, the arbitrary fields wanted to keep after the update must be present in the request, otherwise they will be removed.

Security

Only the administrator and the user itself can access this method.

Content-Typeapplication/vnd.kii.UserUpdateRequest+json
