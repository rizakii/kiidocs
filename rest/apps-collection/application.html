This resource represents an application in the Kii platform.
Location/apps/{appID}
  • GET
  • POST

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: No

GET

/apps/{appID}

Description

Retrieve the properties of the application being accessed.

Security

The access to this operation is not restricted. But only all properties will be recovered if accessing as administrator, if not, only "appID" and "description" properties will be retrieved.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
URL Params
Name Required? type Description
appIDYesAppID of the application being accessed
Response
200

Content-Type application/vnd.kii.AppRetrievalResponse+json

Successfully retrieved the application properties

Content
NameType Description
appIDstringThe ID of the application
descriptionstringA description of the application
emailAddressstring The e-mail address that will be used for any outgoing e-mail coming from this application. Email address length must be less than 200 characters.
emailAddressVerificationRequiredboolean Indicates if the users of the application have to validate their e-mail address
phoneNumberVerificationRequiredboolean Indicates if the users of the application have to validate their phone number
smsFromstring The sender text for the SMSs to send for this application by the platform
verificationSmsTemplatestring The template to use in the SMSs to send to the users to do the phone number verification
facebookAppIDstring The ID of the application in Facebook
exposeFullUserDataToOthersboolean Indicates if the application exposes all of user data to other users in the application. If not, only "userID", "loginName" and "displayName" user properties will be exposed
gcmCollapseKeyDefaultBehaviorstring Configure the "collapse key" that will be sent in the push messages originated by events with GCM. Possible values are "NONE" and "SEND_BUCKET_ID". "NONE" implies collapse key will no be set, and "SEND_BUCKET_ID" implies the "bucketID" where the event is originated will be set as collapse key, this last is the default behavior.

POST

/apps/{appID}

Description

Modify the properties application being accessed.

Security

Can be only performed with administrator credentials.

Content-Typeapplication/vnd.kii.AppModificationRequest+json

The request with the properties to modify in the application
