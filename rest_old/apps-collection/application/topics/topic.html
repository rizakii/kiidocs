This resource represents a topic in an application, user or group (depending on the scope).
Location/apps/{appID}/topics/{topicID}
  • PUT
  • GET
  • DELETE

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: Yes

PUT

/apps/{appID}/topics/{topicID}

Description

Create a new topic. The topicID must be an string following the pattern [A-Za-z0-9_-]* and with maximum length of 64 characters.

Security

This action can be performed with administrator credentials, if not, the permission to perform this action depends on the ACL of the application, user or group, where the topic belongs.

DELETE

/apps/{appID}/topics/{topicID}

Description

Delete the topic with all the subscriptions it has.

Security

Can be performed by the administrator, the creator of the topic and the scope owner (the group owner if the topic is in a group or the user owner if the topic belongs to a user).
