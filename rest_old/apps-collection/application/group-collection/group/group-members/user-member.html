PUT

/apps/{appID}/groups/{groupID}/members/{userID}

Description

Add the current member to the group.

Security

Only can be performed by the administrator and the group owner.

Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app

AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
URL Params
Name Required? type Description
userIDYesxml:stringThe user ID of the member

200
