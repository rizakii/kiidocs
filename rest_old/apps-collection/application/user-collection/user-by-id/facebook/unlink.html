This resource allows to unlink the accessed user from a Facebook account.
Location/apps/{appID}/users/{userID}/facebook/unlink
  • POST

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: Yes

POST

/apps/{appID}/users/{userID}/facebook/unlink

Description

Unlink the accessed user from Facebook.

Security

Only the administrator and the user itself can access this method.

Headers
Name Required? type Description
X-Kii-AppIDYesxml:stringThe AppID assigned by Kii for the current app
X-Kii-AppKeyYesxml:stringThe AppKey assigned by Kii for the current app
AuthorizationYesDescription : Authorization header. Only Oauth2 Bearer token supported for now.
Response
200

Content-Type

The user has been unlinked successfully.

404

Content-Type application/vnd.kii.UserNotFoundException+json

The user to unlink is not found.

Content
NameType Description
errorCodestringError code USER_NOT_FOUND
messagestringThe error message
fieldstring The field used to search for the user, can be the userID field or an address field
valuestring The value of the field used to look for the user
appIDstring The appID of the application where the user belongs
401

Content-Type application/vnd.kii.UnauthorizedAccessException+json

Not authorized to unlink the user.

Content
NameType Description

409

Content-Type application/vnd.kii.UserNotLinkedException+json

Content
