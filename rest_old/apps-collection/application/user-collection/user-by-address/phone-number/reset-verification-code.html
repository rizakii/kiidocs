--- layout: rest-api title: 'Reset Verification Code' tags: rest-api ---
This resource allows to reset the verification code for the phone number of the user.
Location/apps/{appID}/users/{accountType}:{address}/phone-number/reset-verification-code
  • POST

Request Headers (Applies to all methods)

X-Kii-AppID
header
Required: Yes
X-Kii-AppKey
header
Required: Yes
Authorization
header
Required: Yes

POST

/apps/{appID}/users/{accountType}:{address}/phone-number/reset-verification-code

Description

Reset the verification code for the phone number and retrieve it.

Security

Can only be performed with administrator credentials.
