This resource represents a verb (permission) that can be granted or revoked to a subject

GET

/apps/{appID}/users/{accountType}:{address}/acl/{ACLVerb}

Description

Retrieve the ACL entries for the current verb.

Security

The administrator always can perform this action and depending on the resource can also do it: <ul> <li>User: the user.</li> <li>Group: the group owner.</li> <li>Bucket: the scope-owner and the bucket creator.</li> <li>Object: the scope-owner and the object owner.</li> <li>Topic: the scope-owner and the topic creator.</li> </ul>
